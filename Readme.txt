🎬 Netflix Landing Page Clone

A responsive and visually engaging Netflix landing page clone built with **HTML**, **Tailwind CSS**, and **SwiperJS**. This project replicates the core look and feel of Netflix's front-end interface, showcasing trending content with interactive sliders and responsive design.


🔥 Features

🎥 Hero Banner with featured content and call-to-action buttons
🎞️ Interactive sliders powered by SwiperJS for:

   Trending Now
   Popular on Netflix
   Award-Winning Shows
⚡ Fully Responsive Design for mobile, tablet, and desktop
🌙 Dark Theme inspired by Netflix UI
🧭 Intuitive navigation bar** with logo and search icon
📦 Optimized with Tailwind CSS for utility-first styling
🖼️ Image hover effects for enhanced user experience


🛠️ Technologies Used

| Technology       | Purpose                          |
| ---------------- | -------------------------------- |
| **HTML5**        | Page structure                   |
| **Tailwind CSS** | Styling & responsive layout      |
| **SwiperJS**     | Carousel/slider functionality    |
| **Font Awesome** | Icons (e.g., play, search, info) |
